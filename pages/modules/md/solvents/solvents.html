solvents¶

Contains helper functions for initiating SolventConfig subclasses for solvents

It has two public functions, get_solvent_names and get_solvent_config, for initating SolventConfig classes. It has a number of private functions for finding SolventConfig subclasses and WaterModel subclasses which can be used as solvents.
