force_field_factory¶

Factory class for generating force fields

class MDMC.MD.force_fields.force_field_factory.ForceFieldFactory[source]¶

Provides a factory for creating force fields. Any facade within the force fields folder can be created with a string of the class name, as long as it is a subclass of ForceField.
